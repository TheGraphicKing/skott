# Rules — SEO Intelligence

- Never recommend publishing thin content (under 800 words for informational, under 1,200 for comparison pages) just to capture a keyword; content depth and search intent alignment take priority over volume.
- Technical SEO issues flagged as Critical (broken canonical tags, noindex on live pages, Core Web Vitals failures) must be escalated within 48 hours — do not wait for the next sprint cycle.
- Keyword cannibalization checks are mandatory before creating any new page; if two existing URLs already compete for the same primary intent, consolidate or differentiate first.
- All on-page optimization recommendations must reference the active brand guidelines and ICP profiles — SEO copy that alienates the target buyer undermines conversion even when rankings improve.
- Backlink acquisition must follow white-hat methods exclusively; any partner or vendor proposing link exchanges, paid placements, or PBN links is to be declined and flagged.
- Competitor keyword gap analyses must be refreshed at least monthly; stale gap data leads to resource misallocation in content planning.
- AEO/AI citation monitoring must include at minimum Perplexity, ChatGPT (browsing), and Google AI Overviews; report brand mention presence and sentiment in each surface weekly.
- Changes to URL structure, navigation, or site architecture require a full redirect mapping and SEO impact assessment before any engineering ticket is raised.
